## 其它

- 替换微软内置码表

替换掉win10内置的五笔码表，改为98版码表。使用方法见包内详情。

- 小狼毫98五笔版

``````
全功能字体版，在win7和win10下都有较好的渲染效果，推荐使用。
非全功能字体版，采用字体链接的方式实现。
F4切换「方案选单」，有「单字」和「含词」两种98五笔方案。
「含词」部分，采用「高频单字-词组-生僻字」的排序模式。
``````

- 小小98五笔
``````
小小输入法在win下的实现，兼容性比较理想。
``````

- 柚子98版
``````
解压到U盘中，随时使用。
``````
